## PROJETO MarketPlace <br>
Este é um código Python para um MarketPlace, onde os usuários podem realizar compras de sementes. O código solicita informações de login, realiza o cadastro do usuário, permite a escolha de um plano de assinatura, adiciona sementes ao carrinho de compras, calcula o total da compra, solicita a escolha de um método de pagamento e exibe uma mensagem de agradecimento.

## Instruções de uso<br>
1-Ao executar o código, será exibida uma mensagem de boas-vindas ao MarketPlace.

2-O usuário será solicitado a informar se já possui login. É possível responder "sim" ou "não". Caso a resposta seja "sim", o código solicitará o e-mail e a senha do usuário para realizar a verificação das credenciais. Caso a resposta seja "não", o código seguirá para o processo de cadastro.

3-No processo de cadastro, o usuário será solicitado a fornecer informações pessoais, como nome completo, endereço, CPF, cidade, estado, CEP, e-mail e data de nascimento. Além disso, será solicitado o RM do agricultor para concluir o cadastro.

4-Após o cadastro, o usuário poderá escolher um plano de assinatura. Serão exibidas duas opções: "Plano Padrão" e "Plano Gold". O usuário deve digitar o número correspondente ao plano desejado.

5-Em seguida, o usuário poderá adicionar sementes ao carrinho de compras. O código exibirá as categorias disponíveis (Flor, Hortaliça e Fruta) e solicitará a escolha de uma categoria digitando o número correspondente. Após escolher a categoria, serão exibidas as opções de sementes disponíveis nessa categoria, com seus respectivos preços. O usuário deve digitar o número correspondente à semente desejada.

6-Após escolher a semente, o usuário deverá digitar a quantidade desejada. O código calculará o subtotal da semente selecionada e a adicionará ao carrinho de compras.

7-O usuário terá a opção de continuar adicionando sementes ao carrinho ou encerrar a compra. Para continuar, basta responder "sim" à pergunta. Caso o usuário queira encerrar a compra, deve responder "não".

8-Ao encerrar a compra, o código exibirá as informações das sementes escolhidas, incluindo nome, categoria, quantidade de pacotes e subtotal. Em seguida, será calculado o total da compra, considerando o plano de assinatura escolhido.

9-Após calcular o total da compra, o código exibirá os métodos de pagamento disponíveis: Pix, Boleto, Débito e Crédito. O usuário deve digitar o número correspondente ao método de pagamento desejado.

10-O código solicitará a confirmação do método de pagamento escolhido. Caso o usuário responda "não", ele poderá escolher novamente um método de pagamento. Caso responda "sim", o código exibirá o método de pagamento escolhido.

11-Por fim, o código exibirá uma mensagem de agradecimento pela compra.

## Funcionalidades
1-Verificar login: O programa verifica se o usuário já possui login. Caso positivo, solicita as credenciais de login e valida-as.
2-Cadastro: Caso o usuário não possua login, ele pode se cadastrar informando seus dados pessoais, como nome, endereço, CPF, cidade, estado, CEP, e-mail, data de nascimento e RM (Registro do Mercado).<br><br>
3-Escolha do plano de assinatura: O usuário pode escolher entre dois planos de assinatura: Plano Padrão (gratuito) ou Plano Gold (R$25,00 mensais). O programa solicita a escolha do plano e exibe o plano selecionado.<br><br>
4-Compra: O usuário pode selecionar sementes de diferentes categorias (flor, hortaliça, fruta) e adicionar ao carrinho. O programa exibe as opções de categorias disponíveis, as sementes disponíveis dentro da categoria escolhida e permite selecionar a quantidade desejada. Após cada seleção, o usuário tem a opção de continuar adicionando itens ao carrinho ou encerrar a compra.<br><br>
5-Cálculo do total: Ao encerrar a compra, o programa calcula o valor total a ser pago com base nos itens selecionados e na quantidade de cada item. Se o usuário tiver escolhido o Plano Gold, são aplicados descontos no valor total, dependendo da quantidade total de sementes.<br><br>
6-Método de pagamento: O programa solicita a escolha do método de pagamento entre as opções: Pix, Boleto, Débito e Crédito. O usuário pode confirmar ou alterar a escolha.<br><br>
7-Conclusão: Após a confirmação do método de pagamento, o programa exibe uma mensagem de agradecimento pela compra.<br><br>
